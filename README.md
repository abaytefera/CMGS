 CGMS - Complaint & Grievance Management System
A digital, web-based, and mobile-friendly platform designed for the Environmental Protection Authority. This system allows citizens to submit environmental complaints, track their status in real-time, and enables authorities to manage cases through a structured workflow.



 Tech Stack
Frontend: React.js (v18+)

State Management: Redux Toolkit (RTK) & RTK Query


Styling: Tailwind CSS (Responsive & Mobile-first) 

Routing: React Router DOM


Forms: React Hook Form (with CAPTCHA integration) 


Notifications: Integrated SMS & Email service interfaces 


 Features (Based on System Requirements)
 User Roles 


Citizen/Public: Submit complaints (anonymous option), track status, and view public information.



Handling Officer: Manage assignments and update case progress.



Department Supervisor: Oversight and escalation management.



Admin: System configuration and user management.


Executive Viewer: Read-only access to high-level trends and performance summaries.



 Complaint Lifecycle 

The system tracks complaints through the following stages:


Submitted ➔ Under Review ➔ Assigned ➔ In Progress ➔ Resolved ➔ Closed.
